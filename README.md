# 🧩 Sudoku Pro – Flask Web Application

Sudoku Pro is a modern web-based Sudoku game built using Flask (Python), HTML, CSS, and JavaScript.

It generates random Sudoku puzzles, allows interactive gameplay, and stores player scores using SQLite.

---

## 🚀 Features

- 🎮 Dynamic Sudoku puzzle generation
- 🧠 Real-time input validation
- ⏱️ Timer system
- 💡 Hint feature
- 🌙 Dark mode
- 🏆 Leaderboard system
- 💾 SQLite database integration
- 🎉 Win detection and score saving

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Flask (Python)  
- Database: SQLite  
- Algorithm: Backtracking Sudoku generator
